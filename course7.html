Python for Everybody — Charles Severance

Course Description

This course introduces programming using Python, starting from the basics and progressing to more advanced topics. It is designed for beginners with no prior programming experience.

Students learn how to work with data, automate tasks, and interact with web services and databases using Python.

Instructor Details

Charles Severance is a professor and a well-known educator in the field of programming, particularly Python.

Course Curriculum

  • Python Basics
  • Data Structures
  • File Handling
  • Web APIs
  • Database Interaction
  • Real-world Projects

The course is highly practical and focuses on real-world applications.

Duration

6–8 weeks
