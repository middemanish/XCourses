Full-Stack Web Development with React — HKUST

Course Description

This course provides a comprehensive understanding of full-stack web development using modern technologies. It covers both frontend and backend development, enabling students to build complete web applications.

Students gain experience with React for frontend development and Node.js for backend services, along with database integration and deployment strategies.

Instructor Details

The course is offered by the Hong Kong University of Science and Technology, known for its excellence in engineering and technology education.

Course Curriculum

  • Frontend Development Fundamentals
  • React and Component-Based Architecture
  • Backend Development with Node.js
  • REST APIs
  • Database Integration
  • Deployment and DevOps Basics

The course emphasizes project-based learning and real-world application development.

Duration

8–10 weeks
